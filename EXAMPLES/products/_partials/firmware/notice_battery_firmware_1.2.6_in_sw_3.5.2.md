This software includes the 1.2.6 battery firmware upgrade for 48 V revision 1 batteries in deckload and hook robots. This firmware was first introduced in software 3.5.2. If the battery is not already upgraded to this firmware version or higher, then the firmware upgrade is applied automatically when the robot begins charging.

* You can see which firmware version the battery is running in the robot interface under **Monitoring > Hardware health > Power system > Battery Management System**.

* Do not interrupt the upgrading process or disconnect the charger until the upgrade is done. This takes 2–3 minutes.

* For more information, follow the firmware upgrade procedure described in the [3.5.2 release note](https://supportportal.mobile-industrial-robots.com/software/35/software-35).
