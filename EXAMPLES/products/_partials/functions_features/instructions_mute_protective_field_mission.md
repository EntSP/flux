You can add the Mute protective fields action to a mission:

1. Enable Mute protective fields under **System > Settings > Features**.

  {% image src="/images/mir_interfaces_3.0/settings/mute_protective_fields.webp" alt="Enable the Mute Protective fields action in the interface."  /%}

2. Under **Setup > Missions**, create or edit a mission.
3. Add the action Mute protective fields from the Safety system menu.
4. Edit the action parameters so the Protective fields are muted.

  {% $model %} cannot mute specific Protective fields; you can either mute all or none of the fields. Otherwise, the robot reports an error.

5. Add and drag all the actions you want the robot to execute with muted Protective fields inside the Mute Protective fields action.
