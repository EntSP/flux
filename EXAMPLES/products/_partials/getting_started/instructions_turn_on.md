To turn on the robot, follow these steps:

1. Make sure the battery power source is connected—see {% tagref tag="power_connection" /%}.
2. Press the Power button for three seconds.

    {% image src=$image_power_button_turn_on alt="The location of the Power button on the control panel."  /%}

    {% if $is_forklift %}
    The drive wheel turns to the left and right to correctly center the wheel. This may cause the robot to move slightly.
    {% /if %}

    The status lights waver orange, and the robot starts the software initialization process. When the initialization process ends, the robot goes into Protective stop.

    {% image src=$image_start_up_lights alt="The robot with orange startup lights."  /%}

3. Press the Restart button to clear the Protective stop.

    {% image src=$image_restart_button alt="The location of the Restart button on the control panel."  /%}

4. Turn the Operating mode key to the mode you want to use.
