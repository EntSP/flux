## Front compartment {% tag="front_compartment" %}

1. Unscrew the two screws holding the front cover. Use a T30 screwdriver.

    {% image src="/images/mir250/covers/front_cover_unscrew.webp" alt="Unscrewing front cover" /%}

2. Loosen the bottom corners one at the time by pulling out each corner.

    {% image src="/images/mir250/covers/front_cover_bottom_corners_pull.webp" alt="Loosening bottom corners" /%}

3. Loosen the top corners one at the time by pulling each corner down, and then out.

    {% image src="/images/mir250/covers/front_cover_top_corners_pull.webp" alt="Loosening top corners" /%}

4. Pull off the cover.

    {% image src="/images/mir250/covers/front_cover_remove.webp" alt="Removing front cover" /%}

## Rear compartment{% tag="rear_compartment" %}

1. Push the two white buttons on the rear cover at the same time.

    {% image src="/images/mir250/covers/rear_cover_press_buttons.webp" alt="Pressing rear buttons" /%}

2. Loosen the bottom corners one at the time by pulling out each corner.

    {% image src="/images/mir250/covers/rear_bottom_corners_pull.webp" alt="Loosening rear bottom corners" /%}

3. Loosen the top corners one at the time by pulling each corner down, and then out.

    {% image src="/images/mir250/covers/rear_top_corners_pull.webp" alt="Loosening rear top corners" /%}

4. Pull off the cover.

    {% image src="/images/mir250/covers/rear_cover_remove.webp" alt="Removing rear cover" /%}

## Side compartments

1. Open the front and rear compartments first see {% tagref tag="front_compartment" /%} and {% tagref tag="rear_compartment" /%}.

2. Turn the two screws counterclockwise. Use a T30 bit.

    {% image src="/images/mir250/covers/side_unscrew_with_covers_right.webp" alt="Unscrewing side compartment" /%}

3. Loosen the bottom corners one at the time by pulling out each corner.

    {% image src="/images/mir250/covers/side_corners_no_cover_right.webp" alt="Loosening side bottom corners" /%}

4. Pull off the cover.

    {% image src="/images/mir250/covers/side_remove_no_cover_right.webp" alt="Removing side cover" /%}

## Top compartments

1. Unscrew and remove the antenna from the top cover.

    {% image src="/images/mir250/antenna/antenna_unscrew.webp"   alt="Unscrewing antenna" /%}

2. Unscrew the nut from the antenna connector, and push the antenna cable free of the cover. Use an 8 mm wrench.

    {% image src="/images/mir250/top_plate/antenna_bolt_unscrew.webp" alt="Unscrewing antenna bolt" /%}

3. Unscrew the four screws and remove the cover. Use a T10 bit.

    {% image src="/images/mir250/top_plate/top_unscrew_remove.webp" alt="Unscrewing top compartment" /%}
