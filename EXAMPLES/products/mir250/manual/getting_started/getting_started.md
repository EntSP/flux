# Getting started {% tag="getting_started" %}

To get the robot started, follow these steps:

1.  Unpack the robot—see {% tagref tag="unpack_the_robot" /%}.
2.  Connect the robot to its battery power supply—see {% tagref tag="power_connection" /%}.
3.  Connect to the robot's interface—see {% tagref tag="connect_to_interface" /%}.
4.  Connect the robot to a Wi-Fi network—see {% tagref tag="connect_to_wi-fi_network" /%}.
5.  Drive the robot off the pallet manually—see {% tagref tag="operate_manually" /%}.
