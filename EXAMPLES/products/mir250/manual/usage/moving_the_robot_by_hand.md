# Move the robot by hand {% tag="move_robot_by_hand" %}

Before moving the robot by hand, release the mechanical brakes —see {% tagref tag="manual_brake_release_switch" /%}.

To move the robot by hand, either push or pull it using the corner bumpers or the pull handles under the front and rear covers.

If possible, avoid moving the robot by hand, as it can cause the robot to lose its localization.

{% notice type="notice" %}

Moving the robot by hand forcefully may cause damage to the robot.

* If the robot is stuck, push or pull gently on the corner bumpers or the pull handles to move the robot.
* Do not push or pull the robot sideways.

{% /notice %}

When pushing the robot, only push on the corner bumpers or top cover.

When pulling the robot, use either the front pull handle or the rear pull handle.
