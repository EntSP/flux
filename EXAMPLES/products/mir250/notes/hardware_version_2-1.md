## Description

We are now releasing a new hardware version of MiR250, MiR600, and MiR1350. The new hardware version 2.1 uses a new revision of the 48V battery. There are no other hardware changes.

The new battery has an improved design for manufacturing and longer storage time. Internal components have been updated and the outer casing material revised, resulting in an optimized charging curve, the state of charge is determined more accurately and consistently, and the battery weighs three kilos less than the previous revision.

These minor improvements to the entire battery design result in a more refined version of the battery.

These changes have no effect on the robot's rated charging time, runtime, safety certifications, or other performance metrics. The purpose of this release is to keep the battery models we use in our robots up-to-date.

The new battery is compatible with all past hardware versions of MiR250, MiR600, MiR1350, and the upcoming MiR1200 Pallet Jack.

<image: "battery\_48v\_rev\_2.png">

All MiR250, MiR600, and MiR1350 variants with hardware 2.1 continue to use the same serial numbers and order numbers. Production of hardware version 2.1 robots began on 2024-07-30.

The order number for the new battery as a spare part is changed to 450744. The order number for the previous battery revision is deprecated and can no longer be ordered.

The MiR Battery Service Tool for 48V Batteries is not yet compatible with the new battery revision. You can run a diagnostics check on the new batteries, but due to the internal changes in the battery and the new firmware, the diagnosis may not be accurate. It can both return false positive and false negative results. We will send a release note when the tool is updated and can be used on all 48V batteries.
