# Check information stickers {% tag="check_information_stickers" %}

<image: "mir250\_maintenance\_inspect\_nameplate.png">

## Purpose

To ensure that the unit always can be identified. The information stickers on the robot includes the nameplate, identification label, and the warning label.

MiR250, MiR250 ESD, MiR250 Dynamic, MiR250 Dynamic ESD, MiR250 with traction kit, MiR250 ESD Dynamic with traction kit hardware version 2.0, and MiR250 Base Robot, and MiR250 ESD Base Robot hardware version 2.0 and higher have the nameplate already mounted on the left compartment cover, whereas all hardware 1.0 models came with the nameplate in the box, which should then be mounted as part of commissioning. The factory nameplate can have been replaced with an updated one by the integrator, which includes information about the finished robot application.

If replacing the compartment cover where the nameplate is mounted, make sure to always apply a new nameplate in order to retain compliance with applicable European directives.

See _Nameplate and Identification Label Locations_ for information about the location of these stickers on different robots. You can find this guide on [MiR Support Portal](https://supportportal.mobile-industrial-robots.com/).

Frequency: **yearly**

To complete this task, the robot does not need to be powered.

Turn off the robot, and disconnect the battery.

  * When performing maintenance on robots, you risk electrical shock from electrical wires and connectors if internal components are exposed, crushing, collision, or pinching from unexpected movements.

## Procedure

  1. Check and verify that the nameplate is mounted and intact.

  2. Verify that the identification label is intact and readable. The identification label is mounted on the the robot frame behind the rear compartment cover.

  3. Check and verify that the warning label that came with the robot is mounted and that it is intact and readable.

## Pass criteria <image: "mmg\_pass.svg">

  * All information stickers are mounted, intact, and readable.

## Remediation

If factory stickers have been damaged or lost, you can order new stickers from MiR through your MiR Partner.
