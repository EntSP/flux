# Preparation for  maintenance {% tag="preparation_for_maintenance" %}

This section describes the preparation before performing maintenance of the robot.

## Before you start

  1. Move the robot away from the production environment. Preferably to a dedicated service or maintenance area.

  2. Go to the robot interface, navigate to **System > Settings > Wi-Fi**, and select **Disconnect** to ensure that the robot does not receive commands from MiR Fleet while maintenance is being performed. Note that you will have to reconnect Wi-Fi to perform [Inspect and test the signal lights](Weekly tasks/wm_signal_lights.htm).

  3. Turn off the robot and disconnect the battery as described in [Disconnecting the battery](How to disconnect the battery.htm).

  4. Lift the robot up onto support blocks on a work bench or equivalent for easier access. See [Lifting the robot](How to lift the robot.htm).
