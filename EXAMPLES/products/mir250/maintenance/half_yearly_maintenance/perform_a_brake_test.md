# Perform a brake test {% tag="perform_a_brake_test" %}

<image: "mir250\_braking\_distance.png">

## Purpose

To ensure that the brakes are working as intended. The brake test should be performed as described in MiR Commissioning Guide or _MiR Fleet Enterprise documentation_. You can find these documents on [MiR Support Portal](https://supportportal.mobile-industrial-robots.com/).

It is recommended to document and keep brake test results for future reference.

Frequency: **half-yearly**

See a [video](https://vimeo.com/896158770) of the process on the Mobile Industrial Robots profile on vimeo.com.

## Pass criteria <image: "mmg\_pass.svg">

  * The robot is tested according to the procedure and requirements stated in MiR Commissioning Guide or _MiR Fleet Enterprise documentation_ and passes all tests.
