## Robots covered by this maintenance guide

The MiR250 Maintenance Guide does not apply to MiR top modules for MiR250 robots, but applies to the base robot variants that are part of complete applications, such as MiR250 Hook and MiR250 Shelf Carrier. The MiR250 Maintenance Guide applies to all hardware versions and variants of the MiR250 base robot:

  * MiR250 hardware version 1.0–2.0 (100016)

  * MiR250 ESD hardware version 1.0–2.0 (100018)

  * MiR250 with traction kit hardware version 1.0–2.0 (100026)

  * MiR250 ESD Dynamic with traction kit hardware version 1.0-2.0 (100027)

  * MiR250 Dynamic hardware version 1.0–2.0 (100023)

  * MiR250 ESD Dynamic hardware version 1.0–2.0 (100025)

  * MiR250 Base Robot hardware version 2.0 and higher (100151)

  * MiR250 ESD Base Robot hardware version 2.0 and higher (100152)

Throughout the MiR250 Maintenance Guide, the name MiR250 will be used as a common identifier for all variants, unless otherwise specified.
