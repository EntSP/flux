# Software 3.8.0 {% tag="release_note_3.8.0" %}

## Description

Software version 3.7.3 has been promoted to 3.8.0.

3.8.0 is a minor upgrade that focuses on delivering the latest improvements with an emphasis on stability and reliability. This release is intentionally smaller in scope than typical minor releases, which allows for a more stable and lower-risk upgrade.

This software release includes the new feature Guided move, fixes for MiR250 Shelf Carrier and other issues solved.

## New feature

*   **Guided move - new Move mission action (NEX-8375)**
    
    The Guided move feature is now available, allowing robots to perform guided movements as part of automated processes. With Guided move, you can determine how the robot should get from A to B by defining a set of waypoints the robot must drive through. See _How to use Guided move_ on MiR Support Portal for more information.
