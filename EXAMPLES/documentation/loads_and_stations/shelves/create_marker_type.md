# Create marker types for shelves

Shelves can be transported by robots with MiR shelf top modules, such as MiR600 Shelf Lift, MiR1350 Shelf Lift, or MiR250 Shelf Carrier.

Before creating missions with shelves, you need to define the different marker types that your robot will be docking to. Marker types are used to define what type of shelf is located on a Shelf position and how the robot docks to it, ensuring that the robot picks up the shelf correctly.

If you are using a shelf that has dimensions supported by MiR you can use the corresponding default marker type.

To create a new marker type, follow these steps:

  1. To create a new marker type, go to Maps > Marker types, and select Create.

<image: "create\_marker\_type.png">

  2. Fill in the parameters with the dimensions of your shelf. The parameters are described in [Marker types](../../interface_guide_2.0/setup/marker_types/root.htm).

  3. Select Create when you are done. The marker type is now displayed in the list of marker types.
